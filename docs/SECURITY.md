# Security Policy

## Supported Versions
- Latest tagged release
- Main branch

## Reporting a Vulnerability
- Email: ninadsutrave@gmail.com
- Include reproduction steps, affected version, and impact
- We aim to acknowledge within 72 hours and provide a fix or mitigation plan

## Scope
- Token handling and storage
- Host permissions and webRequest usage
- Data privacy guarantees (no analytics or external transmission)

## Responsible Disclosure
- Do not publicly disclose issues before a fix is available
- Coordinated releases with credit where appropriate
